# Getting Started

Three documents to read before writing your first policy. Start with installation, then core concepts, then the tutorial.

---

| Document | Description |
|---|---|
| [Installation](installation.md) | Which packages to install, peer dependencies, and a verification snippet to confirm everything is working. |
| [Core Concepts](core-concepts.md) | The five key concepts — `PolicyDefinition`, `PolicyRule`, `AuthContext`, `Decision`, and `AuthObserver` — and how they relate. Read this before writing any code. |
| [Your First Policy](first-policy.md) | A step-by-step tutorial building a document access policy from scratch: domain types, rules, evaluation, and an audit observer. |

After completing the tutorial, continue with the [Developer Guide](../developer-guide/README.md) for a deeper treatment of each topic.
